Práctica 8 - WebMap

Esta práctica forma parte del curso básico de Python, dentro del contexto del "Grado en Tecnologías Interactivas" de la Universidad Politécnica de Valencia. En ella, se utiliza la biblioteca Folium para crear mapas interactivos que incluyen elementos como:

  • Capas de calor (heatmaps) para representar densidades de datos geográficos.
  • Marcadores y agrupaciones para visualizar puntos de interés.
  • Exportación de mapas como archivos HTML para su integración en páginas web.

El objetivo principal es proporcionar herramientas visuales para el análisis geográfico y facilitar la creación de mapas personalizados.
