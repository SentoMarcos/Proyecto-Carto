Bienvenido a los detalles del Proyecto 3. Este proyecto se centra en el análisis de datos climáticos utilizando herramientas avanzadas para modelar el cambio climático y sus efectos en el entorno global.

Características principales:

  • Modelos predictivos de cambios climáticos a gran escala.
  • Análisis de tendencias basadas en datos históricos.
  • Interfaz visual para comprender patrones complejos.

El Proyecto 3 combina innovación tecnológica con objetivos de sostenibilidad, proporcionando herramientas clave para afrontar los desafíos climáticos del futuro.

Práctica 3: Cartografía Temática - Raster e Interpolación

En esta práctica se utilizan herramientas de geoprocesamiento raster para crear mapas temáticos mediante interpolación de superficies, utilizando datos de temperatura media y precipitación anual en España.

Objetivos

Crear mapas temáticos de coropletas utilizando herramientas raster, con el fin de representar fenómenos climáticos a través de métodos de interpolación.

Desarrollo

Se utilizan datos de estaciones meteorológicas para crear mapas de temperatura media en junio y precipitación anual en la Península. Los pasos incluyen la creación de capas de puntos, interpolación IDW, recorte por máscara y la configuración de simbología para visualización de los resultados.
