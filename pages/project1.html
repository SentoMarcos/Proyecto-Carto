Proyecto 1

Este documento constituye una introducción al uso del software QGIS en el ámbito de los Sistemas de Información Geográfica (SIG). A través de esta práctica inicial, se establecen las bases para comprender y manejar herramientas fundamentales de procesamiento y análisis de datos geoespaciales.

Objetivos principales:

  • Familiarizarse con la interfaz de QGIS y su estructura funcional.
  • Acceder y gestionar fuentes de información georreferenciada, incluyendo servicios WMS.
  • Visualizar, editar y analizar datos cartográficos en formatos vectoriales y ráster.
  • Crear mapas temáticos con simbología personalizada para representar información geográfica.

Resumen de contenidos:

Durante esta práctica, se introduce el concepto de Infraestructura de Datos Espaciales (IDE) y su importancia en la integración y distribución de datos geográficos. Además, se realiza un recorrido por las principales herramientas de QGIS, incluyendo:

  • Exploración de datos geoespaciales disponibles en organismos oficiales.
  • Importación de capas vectoriales y ráster para su análisis.
  • Aplicación de simbología graduada para destacar patrones en los datos.
  • Generación de mapas finales con elementos como leyendas, escalas y cuadrículas.

Este proyecto inicial sienta las bases para un manejo avanzado del software QGIS, permitiendo a los usuarios desarrollar competencias clave en el análisis y representación de datos geográficos.
